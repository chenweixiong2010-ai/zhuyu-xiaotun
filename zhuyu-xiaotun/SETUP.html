步骤 1：创建 Supabase 项目

访问 Supabase 注册并创建新项目

步骤 2：执行数据库脚本

在 Supabase 控制台进入 SQL Editor，复制并执行 supabase-schema.sql 的全部内容

步骤 3：获取 API 密钥

进入 Settings → API，复制以下两个值：

  • Project URL (例如：https://xxxxx.supabase.co)
  • anon/public key (以 eyJ 开头的长字符串)
步骤 4：配置前端

编辑 config.js 文件，填入您的 Supabase 信息：

const SUPABASE_CONFIG = { url: 'https://你的项目.supabase.co', key: '你的 anon key' };
步骤 5：部署

选择以下任一方式部署：

部署到 Vercel 部署到 Netlify
⚠️ 注意事项：
  • 确保 config.js 中的 URL 和 key 正确无误
  • 首次使用请先在本地测试（python -m http.server 8000）
  • 详细部署指南请查看 DEPLOY.md 文件

祝您使用愉快！如有问题请查看文档。
